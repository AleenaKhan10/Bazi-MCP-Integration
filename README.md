# BaZi MCP + Claude Integration

A web application that generates personalized 13-section BaZi (Chinese Astrology) reports using AI.

## Tech Stack

- **Frontend**: HTML/CSS/JavaScript
- **Backend**: Python (FastAPI)
- **MCP Server**: cantian-ai/bazi-mcp
- **LLM**: Claude API (Anthropic)
- **Output**: PDF + HTML reports

## Project Structure

```
bazi-mcp/
├── backend/          # FastAPI backend
├── frontend/         # Static HTML/CSS/JS
├── mcp-server/       # BaZi MCP Server
└── README.md
```
